# Vue3-SpringBoot-CRUD

For practice and learning purpose.

## Reference

Spring Boot & PostgreSQL - https://www.bezkoder.com/spring-boot-postgresql-example/

Vue CRUD - https://www.bezkoder.com/vue-3-crud/

## Run this project

Springboot:
```sh
cd Springboot
mvn spring-boot:run
```

Vue:
```sh
cd vue-crud
npm i
npm run serve
```
